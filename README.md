# Electron — Offline Oilfield Data Toolkit

[View the public showcase](https://sun6-6-6.github.io/electron-showcase/)

Electron is a privacy-first toolkit for oilfield engineers. It turns
vendor-specific operational chronology files into telemetry, events, settings,
calculations, and engineering insight while keeping the source data on the
user's own computer.

This public repository contains only the project showcase, a privacy-safe cover
image, and an anonymized demonstration video. The complete application source,
operational chronology files, private regression fixtures, well identifiers,
and customer data are not published here.

## What the showcase covers

- One self-contained, approximately 4 MB HTML application
- Fourteen lazily loaded and isolated engineering modules
- Eight supported chronology formats: ARH, OJN, JRN, ZQL, ELR, LOG, ZBK, and IRZ
- Automatic file-format routing
- Telemetry charts, event journals, settings, calculations, and references
- Fully offline operation with no upload or runtime network dependency
